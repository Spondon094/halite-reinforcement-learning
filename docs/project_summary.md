# Project Summary

## Title
**Halite Challenge with Reinforcement Learning**

## One-paragraph summary
This project is a cleaned and modernized version of an academic reinforcement learning experiment built around the Kaggle Halite environment. The work investigates how an agent can learn resource collection and navigation behaviour in a grid-based competitive setting. The repository includes a rule-based baseline agent, a simplified actor-critic training pipeline, and detailed documentation of outputs, design choices, limitations, and future improvements.

## Problem statement
Halite is a sequential decision-making environment where agents must collect resources, return them safely, and manage movement under uncertainty. This makes it a useful testbed for reinforcement learning because success depends on balancing short-term gain and long-term strategy.

## Approach
The project combines:

- a heuristic baseline for interpretable behaviour,
- an actor-critic neural model for action selection and value estimation,
- score-delta-based reward shaping,
- state encoding from the game board,
- documentation-driven refactoring of legacy notebook code.

## Skills demonstrated

- reward shaping
- TensorFlow/Keras implementation
- environment-based experimentation
- code refactoring and project structuring
- technical documentation for reproducibility
